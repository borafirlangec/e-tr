This document contains hot keys for Eurofly 2.

 

Control of plane

These work only while flying.

 

Page up
Toggle the left engine on/off

Page down
Toggle the right engine on/off

Home
Speed up

End
Slow down

Alt+Home
Increase Speed to maximum

Alt+End
Decrease Speed to 0

Delete
Stop increasing or decreasing the speed

Up arrow
Tilt the nose down (up to -30 below zero)

Down arrow
Tilt the nose up (up to 30 above zero)

Right arrow
Turn right by one grade

Left arrow
Turn left by one grade

Turn right by 10 grades

Shift+left arrow

Ctrl+left arrow

Automatic course (only the airplanes within category 2)

The name of the current dispatcher

Say the latest registered airport

0 on the messages panel
Set channel 0 – all world

1 on the messages panel
Set  channel 1 – continent

2 on the messages panel
Set channel 2 – state

3 on the messages panel
Set channel 3 – 300 Km

4 on the messages panel
Set channel 4 – 150 Km

5 on the messages panel
Set channel 5 – 70 Km

6-8  on the messages panel
Set channel 6 7 or 8 – private channel

Spacebar on the messages panel
Toggle Sound of incoming message for channel on/off

 

You can control the airplane (turn, put the nose up and down, set automatic functions or activate flaps, brakes and landing gear) only when the engines are on.

You can activate/deactivate the reverse gear only when the airplane is not moving

You can fuel up and board the passengers only when the airplane is in the parking position or at the gate.
